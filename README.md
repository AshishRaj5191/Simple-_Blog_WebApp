# 📝 Simple Blog App (Flask CRUD Project)

## 📌 Project Overview

This project is a **Simple Blog Web Application** built using **Flask (Python)**. It allows users to perform basic **CRUD operations (Create, Read, Update, Delete)** on blog posts without using any database.

This project is developed as part of the **Web Programming with Python and JavaScript**.

---

## 🎯 Features

* ➕ Create new blog posts
* 📖 View all blog posts
* ✏️ Edit existing posts
* 🗑️ Delete posts
* 🔗 Simple navigation (Home, Create Post)
* 🎨 Basic UI styling using CSS

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML (Jinja Templates)
* CSS

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/ashishraj5191/simple_blog.git
cd simple_blog
```

### 2️⃣ Install dependencies

```
pip install flask
```

### 3️⃣ Run the application

```
python app.py
```

### 4️⃣ Open in browser

```
http://127.0.0.1:5000/
```

---

## 🧪 How It Works

* Blog posts are stored in a **temporary list (in-memory storage)**
* Users can:

  * Add posts using a form
  * Edit existing posts
  * Delete posts
* Flask routes handle all operations
